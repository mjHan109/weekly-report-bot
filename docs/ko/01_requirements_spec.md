# Teams 기반 주간 보고 자동화 요구사항 명세

> **문서 동기화:** `docs/en/01_requirements_spec.md`

## 1. 목표
Teams Bot 주간 보고 자동화. **보고 대상은 팀장 지정.** 미제출 시 **본인 제출 대기** 후 취합. **팀장 대리 제출 없음.**

## 2. 핵심 명령어
| 명령 | 권한 | 설명 |
|---|---|---|
| `보고 대상 지정` | 팀장 | 제출 대상 지정 |
| `이번 주 보고 작성` | 지정 대상 | Task Module (13:00 이전 on-time, **13:00 이후 미제출자 본인 late**) |
| `팀 주간 보고 취합` | 팀장 | late/미제출 있었을 때 **수동** 취합 |
| `팀장 등록` | 관리자·팀장 본인 | 팀장 등록 |

## 3. 사용자 시나리오
### 3.0 보고 대상 지정
팀장이 채널별 제출 대상 저장 → 알림·취합·메일은 지정 대상 기준

### 3.1 개인 보고
- on-time: 13:00 이전 `이번 주 보고 작성` → Task Module → Adaptive Card
- late: 13:00 **미제출자** 본인이 동일 명령으로 제출 (대리 없음)

### 3.1.1 목 10:00 / 3.1.2 목 13:00
- 10:00: 채널 — 미제출자 안내
- 13:00 미제출:
  - 채널 — 미제출자 **본인 작성** 요청
  - **팀장 Adaptive Card** — 미제출 명단, **제출 대기 중**, 메일 불가
- 13:00 전원 on-time → **자동 취합** + 팀장 카드(취합·**메일 작성**)

### 3.1.3 전원 late 포함 제출 완료
- **팀장 Adaptive Card** — **전원 제출 완료 · 취합 가능** + `팀 주간 보고 취합`
- 취합 후 취합 미리보기 카드 → `메일 작성` / `발송`

### 3.2–3.4
- 수정, 메일(전원+취합 후), 팀장 등록
