# Documentation Sync Rule (Always Apply)

- Update **both** `docs/ko/` and `docs/en/` with the same structure and meaning.
- Never commit or mark work complete with only one language updated.
- Phase outputs: `docs/ko/phase_outputs/` and `docs/en/phase_outputs/`.
- ADRs: `docs/ko/adr/` and `docs/en/adr/`.
- When behavior changes, update `01`, `02`, or `05` in both languages.
